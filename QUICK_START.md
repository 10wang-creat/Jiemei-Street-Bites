## 🎯 三步驟上線流程

### 步驟 1: 建立 GitHub 帳號（如果還沒有）
1. 前往 https://github.com
2. 點擊 "Sign up" 註冊
3. 驗證電子郵件

### 步驟 2: 上傳檔案到 GitHub
1. 登入 GitHub
2. 點擊右上角 ➕ → "New repository"
3. Repository 名稱輸入：`sister-menu`
4. 選擇 **Public**（公開才能用 GitHub Pages）
5. 點擊 "Create repository"
6. 點擊 "uploading an existing file"
7. **拖曳所有檔案**（index.html, README.md, DEPLOYMENT.md, .gitignore）
8. **拖曳整個 images 資料夾**
9. 在下方輸入：`Upload menu files with real photos`
10. 點擊 "Commit changes"

### 步驟 3: 啟用 GitHub Pages
1. 在你的 repository 頁面，點擊 **Settings**
2. 左側選單找到 **Pages**
3. Source 選擇：**Deploy from a branch**
4. Branch 選擇：**main** → 資料夾選 **/ (root)**
5. 點擊 **Save**
6. 等待 1-2 分鐘
7. 重新整理頁面，會看到網址：
   ```
   🌐 Your site is live at https://[你的用戶名].github.io/sister-menu/
   ```

## 🔧 常見問題排解

### 問題：圖片不顯示？
**解決方法**：
1. 確認 images 資料夾已上傳
2. 確認檔名完全一致（大小寫、連字號）
3. 清除瀏覽器快取（Ctrl+Shift+R 或 Cmd+Shift+R）

### 問題：網站打不開？
**解決方法**：
1. 確認 repository 是 **Public** 不是 Private
2. 確認 GitHub Pages 已啟用
3. 等待 5 分鐘讓 GitHub 處理
4. 檢查網址是否正確

### 問題：LINE 按鈕沒反應？
**解決方法**：
1. 確認手機已安裝 LINE app
2. 確認 LINE 官方帳號 @sister666 存在
3. 在手機瀏覽器測試（不是電腦）
